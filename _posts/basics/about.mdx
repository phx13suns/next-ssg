## このサイトについて

Next.js を使ってブログを作成したくなったので、雛形としてシンプルなサイトを GitHub Pages に公開することにしました。
このブログではコーディングの備忘録を兼ねて技術的な情報を発信していきます。
なるべく最小構成にしたかったので、コンテンツ管理には DB を使わず Git-based な SSG サイト構成としました。
また、Markdown より MDX のほうが React との相性が良さそうなので MDX から記事を生成しています。

このサイトのソースコードは [GitHub リポジトリ](https://github.com/phx13suns/next-ssg) にて公開中です。
