## このサイトについて

Next.js を使ってブログを作成したくなったので、ひな形としてシンプルなサイトを GitHub Pages に公開することにしました。
このブログではコーディングの備忘録を兼ねて技術的な情報を発信していきます。
なるべく最小構成にしたかったので、コンテンツ管理には DB を使わず Git-based な SSG サイト構成としました。
また、Markdown より MDX のほうが React との相性が良さそうなので MDX を利用して記事を書いています。

このサイトのソースコードは [GitHub](https://github.com/phx13suns/next-ssg) にて公開しています。

### システム構成

- Next.js {packageJson.dependencies.next}
  - App Router
  - Static Exports
  - @next/mdx
- React 18
- Tailwind CSS
- daisy UI
- GitHub Pages
- GitHub Actions
